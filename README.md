# Affiliate Global 🛒

**Your Trusted Source for Tech Deals & Reviews**

Affiliate Global is a modern affiliate marketing website for tech products. Built with pure HTML, CSS, and JavaScript — no frameworks, no backend. Designed to be hosted for free on GitHub Pages.

## Features

- 🎯 **Product Database** — 38+ curated tech products across 6 categories
- 🔍 **Smart Search** — Live suggestions with keyboard navigation
- 🏷️ **Category Browsing** — Filter by brand, category, and deals
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 🛒 **Compare Products** — Side-by-side comparison sidebar with localStorage
- ⏱️ **Countdown Timer** — Dynamic deal banners
- 🌙 **Dark Mode Support** — Respects system preference
- 🎨 **Modern UI** — Inspired by leading e-commerce platforms

## Quick Start

```bash
# Clone the repo
git clone https://github.com/trang/affiliate-global.git

# Open index.html in your browser
# No build step needed — pure static site!
```

Or just visit: **[https://trang.github.io/affiliate-global](https://trang.github.io/affiliate-global)**

## Deployment

This site is designed for **GitHub Pages** (free hosting):

```bash
git push origin main
# Then enable GitHub Pages in repo Settings > Pages
```

## Customization

All product data is in `assets/js/products.js` — add, remove, or modify products easily.

To update affiliate links, edit the `affiliateLink` field in each product entry.
